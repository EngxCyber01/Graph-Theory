1. Introduction to Binomial Coefficients

Definition 1.1: The binomial coefficient C(n, k) or (n choose k) represents the number of ways to choose k elements from a set of n elements without regard to order.

Notation: C(n, k) = (n k) = n! / (k!(n-k)!)

Properties of Binomial Coefficients

  • Symmetry: C(n, k) = C(n, n-k)
  • Boundary Conditions: C(n, 0) = C(n, n) = 1
  • Pascal's Identity: C(n, k) = C(n-1, k-1) + C(n-1, k)
  • Sum Property: Σ C(n, k) = 2n for k=0 to n

Example 1.1:

Calculate C(5, 2):

C(5, 2) = 5! / (2! × 3!)

= (5 × 4 × 3!) / (2 × 1 × 3!)

= 20 / 2 = 10

Answer: There are 10 ways to choose 2 items from 5.

2. Pascal's Triangle

Definition 2.1: Pascal's Triangle is a triangular array of binomial coefficients where each entry is the sum of the two entries directly above it.

Pascal's Triangle (First 6 rows):

                    1
                  1   1
                1   2   1
              1   3   3   1
            1   4   6   4   1
          1   5  10  10   5   1

Each number is C(n, k) where n is the row number (starting from 0) and k is the position in the row (starting from 0).

Example 2.1:

Find the entry at row 4, position 2 in Pascal's Triangle:

This represents C(4, 2) = 4! / (2! × 2!) = 6

We can also verify using Pascal's Identity:

C(4, 2) = C(3, 1) + C(3, 2) = 3 + 3 = 6 ✓

3. The Binomial Theorem

Theorem 3.1 (Binomial Theorem): For any real numbers a and b, and non-negative integer n:

(a + b)n = Σ C(n, k) × an-k × bk for k=0 to n

Example 3.1:

Expand (x + y)³ using the Binomial Theorem:

(x + y)³ = C(3,0)x³y⁰ + C(3,1)x²y¹ + C(3,2)x¹y² + C(3,3)x⁰y³

= 1·x³ + 3·x²y + 3·xy² + 1·y³

= x³ + 3x²y + 3xy² + y³

Example 3.2:

Find the coefficient of x⁵ in the expansion of (2x + 3)⁷:

We need the term where the power of (2x) is 5 and power of 3 is 2:

Coefficient = C(7, 5) × (2)⁵ × (3)²

= 21 × 32 × 9 = 6,048

4. Applications and Important Identities

Key Identities:

  • Vandermonde's Identity: C(m+n, r) = Σ C(m, k) × C(n, r-k)
  • Hockey Stick Identity: Σ C(k, r) = C(n+1, r+1) for k=r to n
  • Sum of Squares: Σ C(n, k)² = C(2n, n)
  • Alternating Sum: Σ (-1)k C(n, k) = 0 for n > 0

Example 4.1: Application to Probability

A committee of 5 people is to be formed from 8 men and 6 women. What is the probability that the committee has exactly 3 men?

Solution:

Ways to choose 3 men from 8: C(8, 3) = 56

Ways to choose 2 women from 6: C(6, 2) = 15

Total favorable outcomes: 56 × 15 = 840

Total possible committees: C(14, 5) = 2,002

Probability = 840/2,002 ≈ 0.420 or 42%

5. Computational Techniques

Efficient Computation Methods:

  1. Dynamic Programming: Build Pascal's Triangle iteratively
  2. Multiplicative Formula: C(n, k) = C(n, k-1) × (n-k+1) / k
  3. Cancellation: Simplify before multiplying large factorials

Example 5.1: Computing C(100, 3) Efficiently

Instead of computing 100!/3!×97!:

C(100, 3) = (100 × 99 × 98) / (3 × 2 × 1)

= 970,200 / 6 = 161,700

Practice Problems

Problem 1:

Prove that C(n, k) = C(n, n-k) using the factorial formula.

Solution

C(n, k) = n! / (k! × (n-k)!)

C(n, n-k) = n! / ((n-k)! × (n-(n-k))!)

= n! / ((n-k)! × k!)

Since multiplication is commutative: n!/(k! × (n-k)!) = n!/((n-k)! × k!)

Therefore, C(n, k) = C(n, n-k) ✓

Problem 2:

Calculate the sum: C(5,0) + C(5,1) + C(5,2) + C(5,3) + C(5,4) + C(5,5)

Solution

Using the identity: Σ C(n, k) = 2n

Sum = 25 = 32

Verification: 1 + 5 + 10 + 10 + 5 + 1 = 32 ✓

Problem 3:

Find the coefficient of x⁴y³ in the expansion of (x + y)⁷.

Solution

In (x + y)⁷, the term with x⁴y³ is:

C(7, 4) × x⁴ × y³

C(7, 4) = 7!/(4!×3!) = (7×6×5)/(3×2×1) = 35

Coefficient = 35

Problem 4:

How many subsets does a set with 10 elements have?

Solution

Each element can be either included or excluded (2 choices per element)

Total subsets = 210 = 1,024

Alternatively: Σ C(10, k) for k=0 to 10 = 210 = 1,024

Answer: 1,024 subsets

Problem 5:

A student must answer 8 out of 10 questions on an exam. How many ways can the student choose which questions to answer?

Solution

This is choosing 8 questions from 10:

C(10, 8) = C(10, 2) (by symmetry)

= 10!/(2!×8!) = (10×9)/2 = 45

Answer: 45 ways

Problem 6:

Verify Pascal's Identity for C(6, 3).

Solution

Pascal's Identity: C(n, k) = C(n-1, k-1) + C(n-1, k)

For C(6, 3):

C(6, 3) = 6!/(3!×3!) = 20

C(5, 2) + C(5, 3) = 10 + 10 = 20 ✓

Identity verified!
